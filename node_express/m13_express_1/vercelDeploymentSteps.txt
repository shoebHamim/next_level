
1. add this file in the root directory:
{
  "version": 2,
  "builds": [
    {
      "src": "dist/server.js",
      "use": "@vercel/node"
    }
  ],
  "routes": [
    {
      "src": "/(.*)",
      "dest": "dist/server.js"
    }
  ],
  "outputDirectory": "dist"
}

2. add "build":"tsc" in package.json->scripts
3. npm i -g vercel
4. vercel login
5. vercel -> follow the steps to deploy
6. add the environment variables 


* to see vercel logs 
use -> vercel logs deployment_url
